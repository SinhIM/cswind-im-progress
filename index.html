Hướng dẫn cài đặt lần đầu / First-time setup
1. Tạo Google Sheet mới → Extensions → Apps Script → dán code (.gs được cung cấp riêng).
2. Chọn hàm setupSheets trên thanh công cụ → bấm Run (cấp quyền nếu được hỏi). Việc này tự tạo các sheet: Progress, MasterData_Projects, MasterData_Pipes, MasterData_Workers, MasterData_Config.
3. Deploy → New deployment → Web app → Execute as: Me → Access: Anyone → Deploy → copy Web App URL.
4. Dán URL vào ô Cấu hình phía trên.
5. Vào sheet MasterData_Projects, MasterData_Pipes (import danh sách Số Ống từ Excel của anh), MasterData_Workers để cập nhật dữ liệu thật — sửa trực tiếp như Excel, không cần đụng code.
